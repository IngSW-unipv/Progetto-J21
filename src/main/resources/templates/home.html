Electronic Ticketing System's Web Application

Progetto-J21

This web application allows both travelers and ticket inspectors to use the electronic ticketing system selecting the specific button from the navigation menu on the top of the screen:

  • Every traveler can buy any type of ticket by clicking on the "Purchase" button
  • The "Ticket Catalog" button provides descriptions of the purchasable travel documents
  • Every ticket inspector can log into the inspection area by clicking the "Verify Validation" button
